# webpack_demo

#### 介绍
基础webpack脚手架

#### webpack 系统要求
Node.js+npm（必须）

npm优化（不影响开发）

#### 功能包括

这份webpack_demo中添加的功能有以下几个点

##### 模块加载器（loader）

加载css、加载less、加载图片、 加载和转译 Vue组件、路由

##### 插件（plugin）

提取css、自动生成HTML、打包前清理旧dist文件

##### 输出（output）

错误追踪、开发服务器、生产环境与开发环境分离
